# vim workshop

## Required software

First, make sure that you have all of these installed:

- GNU Make
- pandoc
- entr
- LaTeX (beamer)

Ubuntu Installation:
```bash
$ sudo apt install pandoc
$ sudo apt-get install texlive-latex-extra
$ sudo apt install entr
```

## Generate a PDF presentation

```
make open
make
```
